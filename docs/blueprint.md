# **App Name**: StreamVerse

## Core Features:

- Movie Upload Panel: React-based UI for users to upload video files, enter titles, and descriptions.
- TUS Integration: Implement TUS protocol for resumable video uploads to Bunny.net.
- Metadata Storage: Store content metadata (title, description, Bunny.net URL) in MongoDB after successful upload.
- API Endpoint for Player URL: Create a Node/Express API to fetch the playable video URL from MongoDB based on content ID or title.
- Content Indexing: Automatically tag the content and improve SEO discoverability

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a modern, cinematic feel.
- Background color: Dark gray (#303030) to emphasize content.
- Accent color: Bright blue (#2196F3) for interactive elements and highlights.
- Font pairing: 'Poppins' (sans-serif) for headlines and short amounts of text, and 'PT Sans' (sans-serif) for body text.
- Use clean, modern icons for navigation and content controls.
- Netflix-style grid layout for content display, with focus on large, appealing thumbnails.
- Subtle transitions and hover effects to enhance user interaction.
